Hero Image Instructions
=======================

Place your homepage hero background photo here.

File naming convention:
  hero-[sitename].jpg

Recommended specs:
  - Minimum size: 1920 x 1080 px
  - Format: JPG, optimized under 400KB
  - Style: Dark or silhouette-style photos work best

Once you add the image, update the src attribute in index.html:
  <img src="assets/images/hero-yourname.jpg" alt="...">
